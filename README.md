## The Game

You are the master of a struggling adventurers' guild. Each turn, a situation lands on your desk — a desperate merchant, a rival guild's provocation, a lord's suspicious request. You make a choice. Resources shift. Reputations change. The world reacts.

Survive long enough, manage your relationships, and complete your arc to write your guild's legend.

### Core Loop

Each run plays as a **narrative card game**:

1. A card presents a **situation** tied to one of your NPCs
2. You pick from **2–3 choices**, each with resource trade-offs
3. Resources (**Gold, Adventurers, Quests, Equipment**) shift — let any hit zero and the run ends
4. An **arc** of milestone events unfolds in the background, building toward a climax
5. **Win** by completing the arc. **Lose** by running out of any resource.

### Features

- **Story-driven cards** — Hundreds of situations across NPCs, factions, crisis events, and world events, each with meaningful choices and branching consequences
- **Arc system** — Every run follows a narrative arc (e.g. *The Bandit War*) with escalating milestones that reward bold play
- **Chained cards** — Choices can trigger follow-up situations days later, creating cause and effect
- **NPC relationships** — Your two chosen companions track loyalty and hostility; treat them well or face repercussions
- **Faction dynamics** — The Thieves' Guild and the Temple have their own agendas; stay neutral or pick sides
- **World events** — Three random world events disrupt every run at fixed turning points, forcing adaptation
- **Legacy traits** — Win a run and unlock a permanent trait that gives your next guild a starting edge
- **Resource tension zones** — When a resource drops low, crisis cards flood the queue; recovery is possible but painful
- **Rumor system** — Some choices leave whispers in the world that resurface as future cards

### Navigation & Options

- **Main menu** with background music and pixel-art atmosphere
- **Save & Continue** — runs auto-save after every card; quit and resume at any time
- **Options panel** — music and SFX volume sliders, mute toggles, standard font accessibility mode
- **In-game overlay** — access options or save & quit without leaving the run

## Tech Stack

| | |
|---|---|
| **Language** | Vanilla JavaScript (ES modules, no framework) |
| **Rendering** | Pure DOM manipulation — no canvas, no React |
| **Styling** | Plain CSS with Angelina custom font |
| **Audio** | Web Audio API via HTMLAudioElement |
| **Persistence** | localStorage (run state + settings + meta-progression) |
| **Tests** | Vitest (balance audit scripts) |
| **Build** | No bundler — runs directly from `index.html` |

## AI Authorship

This project was built entirely through conversation with **Claude Code** over multiple sessions. The AI:

- Designed the game mechanics and card economy from scratch
- Wrote all card content (NPCs, situations, choices, flavor text)
- Built the full engine (card queue, resource system, arc progression, save/load)
- Designed and implemented the UI (navigation shell, options system, CSS theming)
- Self-reviewed and iterated on code quality through automated review loops
